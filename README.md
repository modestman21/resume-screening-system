# resume-screening-system
Screens resume against the given job description
The system processes unstructured PDF/Docx resumes, extracts the text, and applies a machine learning model to predict the job category (e.g., Data Science, Web Development, HR). It helps HR teams prioritize the right talent quickly and accurately.
